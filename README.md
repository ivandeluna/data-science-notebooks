# data-science-notebooks

A collection of Python notebooks covering quantitative methods in economics, statistics, and data science — from experimental design to financial risk modeling.

Each folder corresponds to a topic and contains a self-contained notebook with explanations, code, and visualizations. Notebooks are written to be readable without prior context: theory first, then implementation.

## Contents

| Folder | Topic | Notebook |
|--------|-------|----------|
| `04-econometria-basica` | RCT Design & Difference-in-Differences | `rct_difference_in_differences.ipynb` |

*More notebooks added regularly.*

## How to Use

Clone the repo and run any notebook locally:

```bash
git clone https://github.com/ivandeluna/data-science-notebooks.git
cd data-science-notebooks
```

Or open directly in Google Colab by navigating to any notebook and clicking the **Open in Colab** badge.

## Requirements

- Python >= 3.9
- Core packages: `numpy`, `pandas`, `matplotlib`, `scipy`, `statsmodels`

Install all dependencies:

```bash
pip install numpy pandas matplotlib scipy statsmodels
```
