# 04 — Econometría Básica

Notebooks covering fundamental econometric methods, with emphasis on causal inference and experimental design.

## Notebooks

### `rct_difference_in_differences.ipynb`
**RCT Design & Difference-in-Differences Estimation**

Walks through the two core stages of a Randomized Controlled Trial:
1. **Statistical power analysis** — determining the minimum sample size needed to detect an effect before collecting data.
2. **Difference-in-Differences (DiD)** — estimating the causal impact of a treatment using both manual calculation and OLS regression.

Topics: `causal-inference` · `rct` · `difference-in-differences` · `statistical-power` · `statsmodels`
